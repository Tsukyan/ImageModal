## Image Modal
Utilise :
 - [PHP: Hypertext Preprocessor](http://php.net/downloads.php)
 - CSS3/ HTML5
 - JavaScript
 - [Wampserver](http://www.wampserver.com/)

## Fonctionement

L'image ouvre un modal faisant la taille de l'écran lorsque que l'on clique sur cette dernière.
